# ✅ Test Case

**Test Case ID:** TC-005  
**Title:** Upload Photo to a New Post  
**Requirement ID (if exists):**  
**Module / Area:** Media / Posting  
**Type:** Functional  
**Priority:** High  
**Severity:** Major  

**Preconditions:**  
- User is logged in  
- App has permission to access photos  

**Environment:**  
- Device: iPhone XS 
- OS version:  18.6.2
- App version:  Threads 405.1
- Network: Wi-Fi or Mobile Data

---

## ✅ Test Steps

| Step | Action | Expected Result |
|------|---------|------------------|
| 1 | Open the app | App launches successfully |
| 2 | Tap “Create Post” | Post composer opens |
| 3 | Tap “Add Photo” | Photo picker opens |
| 4 | Select a photo from the gallery | Photo preview appears in composer |
| 5 | Add text | Text displays correctly |
| 6 | Tap “Post” | Photo and text upload successfully |

---

## ✅ Expected Result

- Photo appears in the new post  
- Upload completes without errors  
- Post is visible in feed and profile  

---

## ✅ Postconditions

- New post with photo and text is created and publicly visible
